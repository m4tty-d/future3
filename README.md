# Future 3 prototype

### How to use?

```bash
$ sudo pip3 install numpy matplotlib networkx
```
```bash
$ chmod u+x future.py
```
```bash
$ ./future.py
```

### Used libraries

 * [networkx](https://github.com/networkx/networkx)
 * [numpy](https://github.com/numpy/numpy)
 * [matplotlib](https://github.com/matplotlib/matplotlib)
